Managing security and compliance across multiple cloud environments in Microsoft defender for cloud can result in discrepancies  in the compliance status reports generated from the regulatorz compliance dashboard.

This labs aims to highlight the steps and provide reference documentation how to improve status reports for the Regulatorz compliance dashboard of the Microsoft Defender  for Cloud ensuring that the compliance data is consistently updated and accurately reflects the current state of the cloud resources.

there are three actions to take>
1. Enable continuous export to Log Analytics Workspace
2. Integrate with Microsoft Purview Compliance manager
3. Schedule assessments in every 12 hours
